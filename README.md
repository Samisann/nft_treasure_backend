
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## How to setup a database

``` 
 - select the db folder  with command line
 - run mongosh from this folder (db)
 - tap this command "use nft_treasure" to initdb
 - load file migration via this command < load("v001.js")>

```
